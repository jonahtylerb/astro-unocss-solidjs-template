# Astro Template with Solidjs and UnoCSS

This is the template I use for making websites.


## Features Include
1. Astro
2. UnoCSS
3. Solidjs
4. Astro-Compress
5. Astro-Image-Tools
6. Astro-GoogleFontsOptimizer
7. Typescript
8. Dynamic font size added to UnoCSS
9. Icons with UnoCSS/icons and Tabler
